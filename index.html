web

— desarrollo front end

Tener una página web hoy en día, es esencial para darte a conocer en todo el mundo. La alternativa, significa no existir. 

Por ese motivo estudié desarrollo de aplicaciones web en la Escola Industrial (2014-2016). Aprendiendo como funciona una web por detrás y sobretodo disfrutando de programar la parte visible para el cliente (HTLM + CSS). 

Pero lo importante en una web siempre es el usuario al que va dirigido. Para ello realicé el Master en diseño Web en la BAU, añadiendo valor a las webs que diseñaba con el estudio sobre experiencia de usuario, metodologías agiles, diseño aplicado a web, sass.
